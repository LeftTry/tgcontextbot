Добро пожаловать! Это бот для администраторов телеграм каналов и групп. Он упростит и автоматизирует многие функции полезные в администрировании. На данный момент бот имеет довольно слабый функционал, однако в будущем мы планируем добавить больше возможностей
Основные команды:
/start - команда, запускающая бота, выдает список доступных команд
/help - команда, выдающая список доступных команд
/guide - команда, возвращающая ссылку на гайд

/addblacklist + слова - команда, добавляющая запрещенные слова в базу данных. После этого бот будет сообщать, если данное слово было употреблено
/watchblacklist - команда, позволяющая просмотреть список запрещенных слов для данного чата
/deletefromblacklist + слова - команда, позволяющая удалить выбранные запрещенные слова для данного чата

/setsubstitutewith + слово + || + слово - команда, устанавливающая соответствие между двумя словами. В последствии если первое слово будет употреблено в чате бот вернет слово, с которым это соответствие было установлено.
/deletesubstitute + слово + || + слово - удаляет соответствие между двумя словами в данном чате
/getpairs - возвращает все слова когда-либо употребленные в чате, с которыми было установленно соответствие командой /setsubstitutewith

/addexceptiontosubstitute + слово + || + username + || - устанавливает человека(id) для соответствия двух слов в данном чате
/getexcepted - позволяет посмотреть список всех исключений для соответствия двух слов в данном чате
/deleteexception + слово + || + username + || - удаляет исключение для соответствия двух слов в данном чате

/addbadwordexception + слово + || + username + || - устанавливает исключение для запрещенных слов в данном чате
/getbadwordexceptions - позволяет посмотреть список всех исключений для запрещенных слов в данном чате
/deletebadwordexception + слово + || + username + || - удаляет исключение для запрещенных слов в данном чате
